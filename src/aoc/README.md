
```
aoc
```
Run AoC solutions in the agent mode

<hr>

```
aoc run <year> <day> <part> --input-file <path>
```
Run AoC solution for a specific year, day and part for the input provided.
- `<year>`: Puzzle year
- `<day>`: Puzzle day
- `<part>`: Puzzle part
- `--input-file`: Input file path

<hr>

```
aoc examples <year> <day> <part>
```
Run all examples for a specific year, day and part.
- `<year>`: Puzzle year
- `<day>`: Puzzle day
- `<part>`: Puzzle part
